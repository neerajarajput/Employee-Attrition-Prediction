# Employee Attrition Prediction

## Overview

This project uses Machine Learning to predict employee attrition and identify the factors that contribute to employees leaving an organization. The project includes data preprocessing, model training, hyperparameter tuning, and a Power BI dashboard for business insights.

## Models Used

* Logistic Regression
* Decision Tree Classifier
* Random Forest Classifier
* Tuned Random Forest Classifier

## Key Features

The most important features influencing attrition were:

* Monthly Income
* Age
* Overtime
* Years at Company
* Total Working Years
* Job Level
* Stock Option Level

## Evaluation Metrics

Models were evaluated using:

* Recall
* F1 Score
* Accuracy
* Precision

The Tuned Random Forest model achieved the best overall performance and was selected as the final model.

## Power BI Dashboard

The dashboard provides insights into:

* Attrition Rate
* Department-wise Attrition
* Job Role Analysis
* Overtime Impact
* Employee Tenure Trends
* Income Analysis
* Model Performance Comparison
